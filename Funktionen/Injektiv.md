Injektivität beschriebt das Abbildungsverhalten von Funktionen. Das bedeutet, dass jedes Element der Zielmenge wird höchstens einmal getroffen (kein doppelter y-Wert).

**Definition:** Eine Funktion $f: A \to B$ ist injektiv, wenn verschiedene Elemente aus A immer auf verschiedene Elemente in B abgebildet werden.
**Formel:** $x_{1} \ne x_{2} \implies f(x_{1}) \ne f(x_{2})$
**Bedeutung:** Jedes Element in der Zielmenge B hat **höchstens** ein Urbild in A.
**Beispiel:** $f: \mathbb{R} \to \mathbb{R} \quad mit \quad f(x)=x^3$
**Merkregel:** "Eins-zu-Eins": Keine zwei x treffen denselben y-Wert.
