Surjektivität beschreibt das Abbildungsverhalten von Funktionen. Surjektiv bedeutet, jedes Element der Zielmenge wird **mindestens** einmal getroffen (der Wertebereich entspricht dem Zielbereich).

**Definition:** Eine Funktion $f: A \to B$ ist surjektiv, wenn jedes Element in der Zielmenge B mindestens ein Urbild in A hat.
**Formel:** Für alle $y \in B$ existiert ein $x \in A$, sodass $f(x) = y$.
**Bedeutung:** Jedes Element in der Zielmenge **B** wird **mindestens** einmal getroffen, d.h. der Zielbereich ist vollständig abgedeckt.
**Beispiel:** $f: \mathbb{R} \to \mathbb{R} \quad mit \quad f(x)=x^{3}$ (jeder y-Wert wird erreicht)
**Merkregel:** "Jeder hat zwei Hände"
